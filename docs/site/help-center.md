# Student Help Center Content Plan

The web application will expose help topics as part of the student experience. This document tracks required articles and their owners.

## Sections

1. **Getting Started**
   - Creating an account
   - Selecting your first subject
2. **Practice Sessions**
   - Starting an exam
   - Revealing answers
   - Requesting AI explanations
   - Saving progress
3. **Results & Analytics**
   - Understanding your score
   - Reviewing incorrect answers
   - Tracking progress over time
4. **Account Management**
   - Updating profile
   - Managing notifications
   - Deleting account / data export
5. **FAQ**
   - Common troubleshooting questions
   - Contacting support

## Content Workflow

- Draft articles in Markdown within `apps/web` content directory.
- Review for accuracy and accessibility.
- Publish via CMS module or static Markdown pages.
